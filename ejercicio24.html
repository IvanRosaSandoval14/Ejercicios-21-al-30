Etiqueta SECTION

Un documento web podria dividirse en secciones para un listado de noticias, informacion de contectos, etc.

se usa para crear contenidos independientes dentro de un documento web.

Etiqueta ARTICLE

Un documento web podria contener un numero indeterminado de articulos, ya que cada uno posee independencia frente a los demas articulos inclusive dentro del mismo documento web.

En un articulo podrias colocar etiquetas como HEADER, SECTION Y FOOTER.
